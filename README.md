# Emote IQ

Emote IQ is an AI-based emotion recognition and recommendation chatbot designed to help users understand and manage emotions through text input, image selection, and category-based interaction.

The application uses machine learning and NLP techniques to identify emotions such as joy, fear, anger, and other emotional states. Based on the detected emotion and selected context, Emote IQ provides personalized recommendations such as quotes, audio, and video-based solutions.

## Features

* **Emotion Recognition:** Detects user emotions using machine learning models, including Random Forest.
* **Text-Based Emotion Analysis:** Analyzes user-provided text to identify emotional cues.
* **Image-Based Interaction:** Allows users to select images representing different emotions.
* **Category-Based Recommendations:** Supports contextual categories such as Family, Business, and Education.
* **Personalized Recommendations:** Provides emotion-specific suggestions such as quotes, audio, and videos.
* **User-Friendly Interface:** Offers a simple web interface for chatbot interaction and emotional support.
